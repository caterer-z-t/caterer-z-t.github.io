  <div>
    <p>
      <strong><a href="https://github.com/caterer-z-t/GRiD">GRiD</a></strong> (Genomic Repeat inference from Depth) 
      is a haplotype-length estimator for VNTR and CNV regions from short-read whole-genome sequencing data. 
      GRiD integrates normalized read depth, nearest-neighbor coverage profiles, and haplotype sharing 
      (IBD/IBS) to infer diploid repeat copy number and resolve chromosome-specific repeat lengths.
    </p>

    <p>
      View the source code on <a href="https://github.com/caterer-z-t/GRiD">GitHub</a> or explore the 
      full documentation on <a href="https://caterer-z-t.github.io/GRiD">ReadTheDocs</a>.
    </p>
  </div>
